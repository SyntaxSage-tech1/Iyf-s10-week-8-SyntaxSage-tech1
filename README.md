## Project Description  
This project focuses on building a React-based user interface using Vite. It introduces the core concepts of React including components, JSX, props, and state while developing the foundation of the CommunityHub application.

## Technologies Used  
- HTML5  
- CSS3  
- JavaScript  
- React  
- Vite  

## Features  
- Component-based architecture  
- Dynamic rendering using JSX  
- Passing data through props  
- State management for interactive UI  
- Event handling for user actions  
- Controlled forms for user input  
- Conditional rendering of UI elements  
- Organized and reusable component structure  

## How to Run  
1. Clone this repository  
2. Navigate to the project folder  
3. Run `npm install`  
4. Run `npm start` to launch the app  

## Lessons Learned  
I developed a solid understanding of React fundamentals such as components, props, and state. I also learned how to handle events, manage form inputs, and structure a React application effectively.
